## 🚀 DnsThief

> **Fast, lightweight and effective subdomain brute-force tool for reconnaissance.**

DnsThief is a high-performance DNS enumeration tool built for speed and simplicity. Designed for pentesters, bug bounty hunters, and security researchers, it brute-forces subdomains using customizable wordlists and concurrent resolution.

With a clean CLI interface and multithreaded execution, DnsThief allows you to quickly uncover hidden attack surfaces across target domains.

---

### 🔥 Features

- ⚡ **Fast multithreaded scanning** (configurable concurrency)
- 📚 **Built-in smart wordlist** (no setup required)
- 🧩 **Custom wordlist support**
- 🎯 **Flexible input** (CLI or stdin)
- 🔇 **Silent mode for automation & pipelines**
- 🧠 **Efficient DNS resolution using sockets**
- 🪶 **Lightweight & dependency-free**
